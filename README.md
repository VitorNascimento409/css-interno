# CSS Interno

Este é um projeto simples desenvolvido com o objetivo de praticar estruturação de páginas HTML e estilização utilizando CSS interno.

## Descrição
O projeto consiste em uma página web estática, onde o estilo é aplicado diretamente no arquivo HTML por meio da tag `<style>`.  
O foco foi compreender como o CSS interno funciona e como ele influencia a aparência dos elementos da página.

## Tecnologias utilizadas
- HTML5  
- CSS interno  

## Objetivo
- Praticar a estrutura básica do HTML  
- Entender o funcionamento do CSS interno  
- Aplicar estilos como cores, fontes, espaçamentos e alinhamentos  

## Observações
Este foi mais um projeto desenvolvido para fins de estudo.  
Em projetos como esse que utiliza somente de uma página para estilização é vantajoso utilizar o CSS interno, já em projetos maiores, o uso de CSS externo é mais indicado para melhor organização e manutenção do código.
